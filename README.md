# University Admission & Academic Management System

A Core Java application that simulates university admission, branch allocation, course assignment, and academic record management.

## Features
- Branch allocation based on competitive exam marks
- Dynamic course assignment per branch
- Student record management
- Academic analytics (highest CGPA)

## Technologies Used
- Java
- Java Collections (ArrayList, HashMap)
- Object-Oriented Programming

## How to Run
1. Compile the program:
   ```bash
   javac Working.java
2. Run the program:
java Working

## Project Structure
- Student.java – Stores student details
- Course.java – Manages branch-wise courses
- Enrollment.java – Handles branch allocation logic
- UniversityManagement.java – Core system logic
- Working.java – Entry point and user interaction
